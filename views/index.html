Timestamp Microservice event_note

User Story

1. I can pass a string as a parameter, and it will check to see whether that string contains either a Unix timestamp or a natural language date (e.g., January 1, 2016)
2. If it does, it returns both the Unix timestamp and the natural language form of that date.
3. If it does not contain a date or Unix timestamp, it returns null for those properties.

Example Usage

Enter the date in the url bar in the UTC (yyyy-mm-dd) or UNIX format

  • /api/2022-12-25
  • /api/1671926400000

Example Output

It will give an output like:
{ "unix":1451001600000, "utc":"Fri, 25 Dec 2015 00:00:00 GMT" }
